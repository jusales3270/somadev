# Projeto (SomaDev) — README

> Documentação inicial do projeto + guia de setup do Supabase.

## Visão geral
Este repositório contém a base do projeto no ecossistema **SomaDev**.  
A aplicação utiliza **Supabase** para autenticação, banco de dados (Postgres), storage e/ou edge functions (conforme o uso do projeto).

## Requisitos
- Node.js (recomendado: LTS)
- npm / pnpm / yarn (um gerenciador de pacotes)
- Conta no Supabase (para ambiente cloud) **ou** Supabase CLI (para ambiente local)
